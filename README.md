# Streak Garden

A streak tracker where you can see your progress through your garden

An app where you track your streaks and they show up as plants in a garden. The longer your streak, the bigger the plant grows.
- Track different streaks (like sobriety, working out, )
- Show them as plants
- Give you seeds for keeping streaks going
- Let you buy decorations with seeds

## Tech
- Python
- Streamlit

## To run it
```bash
streamlit run streak_garden.py
```
